# Xfetch — Landing Page

Landing page for the [xscriptor/xfetch](https://github.com/xscriptor/xfetch) repository. Cross-platform system information fetching tool written in Rust, inspired by fastfetch and neofetch. Dark terminal-inspired design with interactive terminal demo and config builder.

## Propósito

Ejercicio personal de diseño y desarrollo front-end. La landing presenta el proyecto xfetch con un simulador de terminal interactivo, constructor de configuración en vivo, showcase de layouts, y documentación completa del ecosistema de plugins y módulos.

Tecnologías: Next.js 16, Tailwind CSS 4, TypeScript.
