# Consultoria.io — Rediseño

Rediseño del sitio web de Consultoria.io, consultoría especializada en ecommerce. El proyecto reconstruye la plataforma con un enfoque en la conversión, la credibilidad y la experiencia de usuario profesional.

## Propósito

Ejercicio personal de rediseño y reconstrucción front-end. El sitio original está construido sobre HubSpot; esta versión moderniza la interfaz con tecnologías actuales optimizando rendimiento, accesibilidad y mantenibilidad.

Tecnologías: Next.js 14, Tailwind CSS, TypeScript.
