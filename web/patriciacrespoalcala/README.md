# Patricia Crespo Alcalá — De Poéticas

Rediseño del sitio web personal de Patricia Crespo Alcalá, poeta y crítica literaria. El proyecto reconstruye su presencia online con un enfoque en la experiencia de lectura, la tipografía y la narrativa visual.

## Propósito

Ejercicio personal de rediseño y reconstrucción front-end. El sitio original funciona como bitácora literaria; esta versión moderniza la interfaz preservando la identidad poética y el tono editorial.

Tecnologías: Next.js 14, Tailwind CSS, TypeScript.
