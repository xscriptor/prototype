# Terminal Xscriptor — Landing Page

Landing page for the [xscriptor/terminal](https://github.com/xscriptor/terminal) repository. Showcases 12 color palettes configured for 27 terminal emulators and 4 prompt engines, with a dark-first design and animated background paths.

## Propósito

Ejercicio personal de diseño y desarrollo front-end. La landing presenta los themes del repo de terminales con un enfoque visual oscuro, tarjetas glassmorphism, paletas interactivas y paths SVG animados con framer-motion.

Tecnologías: Next.js 16, Tailwind CSS 4, TypeScript, framer-motion.
