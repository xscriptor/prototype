# Prototype — Colección de Interfaces Rediseñadas

Este repositorio alberga una colección de sitios web y prototipos de interfaz que he rediseñado por mi cuenta como parte de mi práctica y desarrollo profesional en front-end e ingeniería de interfaces.

## Propósito

Cada proyecto dentro de `web/` es un **ejercicio personal de rediseño**. Tomo sitios web existentes (públicos, de marcas, servicios o proyectos personales) y los reconstruyo desde cero usando tecnologías modernas como Next.js, Tailwind CSS, TypeScript, etc. El objetivo es:

- Demostrar capacidad técnica en el desarrollo de interfaces completas
- Explorar decisiones de diseño, arquitectura de componentes y rendimiento
- Construir un historial público de trabajo en front-end e ingeniería web
