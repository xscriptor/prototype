# Prototype — Redesigned Interface Collection

This repository hosts a collection of redesigned websites and UI prototypes built independently as part of my professional practice in front-end engineering and interface development.

## Purpose

Each project under `web/` is a **personal redesign exercise**. I take existing public websites (brands, services, or personal projects) and rebuild them from scratch using modern technologies such as Next.js, Tailwind CSS, TypeScript, and more. The goals are:

- Demonstrate technical proficiency in building complete interfaces
- Explore design decisions, component architecture, and performance
- Build a public portfolio of front-end and web engineering work
